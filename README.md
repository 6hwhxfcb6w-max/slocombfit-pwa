# Slocomb Fitness Center — PWA

Progressive Web App for Slocomb Fitness Center in Slocomb, Alabama.

**Live Local. Lift Local.**

## Features

- 📋 Class schedules with real-time countdown
- 🏋️ Structured workout generator (30 & 60 min routines)
- 📢 Announcements & notifications
- 👨‍🏫 Trainer profiles & management
- 🌙 Light/Dark mode
- 📱 Installable PWA with offline support
- 🔐 Admin panel for gym staff
- 💬 Member feedback system

## Tech Stack

- React 18
- CSS-in-JS theming
- Service Worker for offline support
- PWA manifest for installation

## Getting Started

```bash
npm install
npm start
```

## Deployment

Built for Cloudflare Pages with automatic deploys from GitHub.

```bash
npm run build
```

Build output: `build/`
